# AKD Expiry Bot

AKD Expiry Bot — Telegram-бот AKDworks для отслеживания сроков действия документов, услуг и обязательств.

Telegram username: `@akdexpiry_bot`

## О Проекте

Бот помогает хранить список важных сроков и заранее получать напоминания.

Примеры того, что можно отслеживать:

- паспорт;
- водительские права;
- страховку;
- техосмотр;
- банковскую карту;
- домен;
- SSL-сертификат;
- хостинг;
- лицензию на софт;
- сертификат;
- медсправку;
- договор;
- пропуск.

## Какие Данные Хранит Бот

Для работы напоминаний достаточно хранить:

- Telegram ID пользователя;
- название записи;
- категорию;
- дату окончания или месяц окончания;
- заметку по желанию;
- настройки напоминаний.

## Возможности

- команда `/start`;
- добавление записи;
- выбор категории;
- точная дата окончания в формате `ДД.ММ.ГГГГ`;
- месяц окончания в формате `ММ.ГГГГ`;
- заметка по желанию;
- выбор напоминаний: за 30 дней, за 7 дней, за 1 день и в день окончания;
- список записей пользователя;
- просмотр одной записи;
- редактирование записи;
- удаление записи с подтверждением;
- автоматические напоминания;
- кнопка `О боте`;
- локальное хранение данных в SQLite.

## Инструменты

- Python;
- aiogram 3;
- SQLite;
- APScheduler;
- python-dotenv.

## Конфиденциальность

Бот не должен использоваться для хранения номеров документов, персональных идентификаторов, полных адресов, данных банковских карт и другой чувствительной информации.

Достаточно короткого названия, категории, даты окончания и необязательной заметки.
